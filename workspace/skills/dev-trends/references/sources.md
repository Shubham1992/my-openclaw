## Notes

- GitHub Trending is the most reliable source for project-level trends
- HN skews toward systems programming, startups, and AI
- Dev.to skews toward web dev and beginners
- Reddit r/programming skews senior/experienced developers
- Combine 2–3 sources for a balanced view
